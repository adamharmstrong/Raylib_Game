# First-Party Assets

Use this folder for assets created by the project team.

- `characters/`: player, NPCs, and character animation source files.
- `environments/`: original tiles, props, and background art.
- `machines/`: winches, pulleys, gears, doors, gates, and mechanical parts.
- `ui/`: menus, icons, fonts, and HUD art.
- `audio/`: music, ambience, and sound effects.
